# Lab overview
This lab demonstrates a basic Static Routing configuration.

## Topology includes
- 3 Routers
- 3 Switches
- 3 PCs
- 1 Laptop
- 1 Server

## Network structure:
- 3 Local Area Networks (LANs)
- 2 Point-to-Point WAN links between routers.
